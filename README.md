# Banking System in Java

A simple console-based Banking System project developed using Java.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Account Details
- Display All Accounts

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner Class

## Run Project

```bash
javac BankingSystem.java
java BankingSystem
```
